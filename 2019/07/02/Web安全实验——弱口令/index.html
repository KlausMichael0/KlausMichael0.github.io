Web安全实验——弱口令

Posted on 2019-07-02

弱口令指的是仅包含简单数字和字母的口令。

后台弱口令发现与攻击实验

一. 实验目的

1.了解弱口令的危害；

2.了解弱口令攻击的原理和方法；

3.学会X-Scan的使用。

二. 实验原理

弱口令(weak password) 没有严格和准确的定义，通常认为容易被别人（他们有可能对你很了解）猜测到或被破解工具破解的口令均为弱口令。弱口令指的是仅包含简单数字和字母的口令，例如“123”、“abc”等，因为这样的口令很容易被别人破解，从而使用户的计算机面临风险，因此不推荐用户使用。

三. 实验内容

1.介绍弱口令的含义及危害 2.介绍弱口令的攻击原理及方法 3.掌握X-scan工具的使用方法

四. 实验环境描述

1、 学生机与实验室网络直连； 2、 VPC与实验室网络直连； 3、 学生机与VPC物理链路连通；

pc机：windows7旗舰版 ip：172.16.1.156

五. 实验步骤

注：实验中所有ip(如：172.17.135. 或 110.10.10.)均要根据实际网络情况而改变，所用工具windows版的均在D:/tools中，linux版的均在 桌面 上。

一、 单击实验环境，单击实验环境，点击打开控制以启动网络拓扑中的电脑（即虚拟机）

二、 win7默认用户名：admin，密码：123456

三、 步骤1：进入实验虚拟机，在“控制面板”中关闭Windows自带防火墙，如下图所示：

步骤2：首先在控制面板中打开程序，添加windows功能找到telnet服务，打开在“服务”中开启Windows的Telnet服务，如下图所示：

步骤4：点击FileZilla服务的“Admin“，进入管理面板，如下图所示：

步骤5：点击“Edit”-“Users”，在配置面板中添加一个弱口令用户，如“test/123456”,在“Shared folders”菜单中配置FTP路径，如下图所示：

步骤6:打开X-Scan软件，进入配置界面，在“指定IP范围”内输入虚拟机的IP地址，本例为“192.168.192.128”，如下图所示：

步骤7：在“全局设置”-“扫描模板”中勾选“开发服务”，“TELNET弱口令”，“FTP弱口令”，如下图所示：

步骤8：在“全局设置”-“其他设置”中选择“无条件扫描”，并勾选“显示详细进度”，点击“确定”保存，如下图所示：

步骤10：打开X-Scan主界面上点击“开始扫描”，则软件会开始扫描弱口令，如下图所示：
